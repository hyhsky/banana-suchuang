# Banana AI - 智能图像生成工具

基于 Nano Banana Pro (Gemini 3 Pro) 模型的现代化图像生成应用。

## 🚀 如何运行

### 方法 1：一键启动 (推荐)
直接双击项目目录下的 **`启动应用.bat`** 文件即可。

### 方法 2：命令行启动
如果您是开发者,可以使用标准命令:
```bash
# 安装依赖
npm install

# 启动服务器
npm run dev
```

## ✨ 功能特性
- **文本生成图像**: 支持 4K 超高清分辨率
- **图像生成图像**: 支持上传参考图进行风格迁移
- **多图融合**: 支持多张图片混合创作

## 🛠️ 技术配置
- **模型**: NanoBanana-pro (Nano Banana 2)
- **API**: 速创 API (https://api.wuyinkeji.com/)
